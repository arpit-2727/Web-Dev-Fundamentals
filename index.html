Web Development Documentation

Web Development

  • refers to the creating, building, and maintaining of websites
  • includes aspects such as web design, web publishing, web programming, and database management.
  • It is the creation of an application that works over the internet i.e. websites.

Types of Web Development

  • Front End Development
  • Back End Development
  • Full Stack Development

Core Concepts

  • HTML
  • CSS
  • Javascript

HTML

  • Hyper Text Markup Language
  • is used to design the web pages
  • create a complete website structure

CSS

  • Cascading Style Sheets
  • is used for styling the webpages
  • helps in making the webpage visually appealing

Javascript

  • JavaScript is a programming language
  • It allows you to make your page interactive
  • It can be used to add effects like animations or slideshow on the webpages
